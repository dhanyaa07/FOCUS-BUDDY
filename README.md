# FOCUS-BUDDY
🧠 ADHD Assist

ADHD Assist is an AI-powered web application developed as a mini project to support children aged 5–12 years diagnosed with Attention Deficit Hyperactivity Disorder (ADHD). The platform aims to improve attention, focus, emotional regulation, and daily routine management through engaging, child-friendly activities and AI-assisted monitoring.

The application provides interactive cognitive training games, CBT (Cognitive Behavioral Therapy)-based activities, AI-powered emotion detection, bedtime stories, and guided yoga videos that children can practice independently to encourage mindfulness and relaxation. It also features smart reminders, progress tracking, and role-based access for children, parents, teachers, and therapists, enabling continuous monitoring and personalized support.

The project was developed after consulting a child psychiatrist, whose feedback helped shape the activities and features to ensure they are age-appropriate, engaging, and beneficial for children with ADHD.

✨ Key Features
🤖 AI-powered emotion detection
🧩 Cognitive training & CBT-based games
📚 Bedtime stories for relaxation and better sleep
🧘 Guided yoga videos for independent practice
📊 Progress tracking and performance analytics
⏰ Smart reminders for daily routines
👨‍👩‍👧 Role-based access for Child, Parent, Teacher & Therapist
🔒 Secure authentication and data management
🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: SQLite
AI: Face API.js (Emotion Detection)
Authentication: JWT, bcrypt.js
